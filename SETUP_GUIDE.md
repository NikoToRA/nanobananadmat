## ✅ 確認済み：画像ダウンロードボタン

画像ダウンロードボタンは既に実装済みです！
- 生成された画像の下に「💾 画像をダウンロード」ボタンが表示されます
- クリックすると、各ユーザーのPCに画像がダウンロードされます

## 📋 必要なもの（これだけ！）

### 1. Gemini APIキー（必須）

**取得方法：**
1. [Google AI Studio](https://makersuite.google.com/app/apikey)にアクセス
2. 右上の「Get API Key」または「Create API Key」をクリック
3. APIキーが生成される（例: `AIza...`）
4. コピーして保存

**費用：** 無料枠あり（60 requests/分まで）

**注意：** Gemini APIは画像生成をサポートしていない可能性があります。動作しない場合は、下記のReplicate APIを使用してください。

### 2. Replicate APIトークン（オプション・推奨）

Gemini APIが画像生成をサポートしていない場合の代替案です。

**取得方法：**
1. [Replicate](https://replicate.com)にアカウント作成（GitHubアカウントでログイン可）
2. 右上のプロフィールアイコン → 「API Tokens」
3. 「Create token」をクリック
4. トークンをコピーして保存

**費用：** 無料枠あり（最初の$5分無料）

**推奨理由：**
- Stable Diffusionを使用（高品質な画像生成）
- 確実に動作する
- 無料枠が充実

### 3. GitHubアカウント（デプロイ用・オプション）

Webデプロイする場合に便利ですが、必須ではありません。
- [GitHub](https://github.com)でアカウント作成（無料）

### 4. Vercelアカウント（デプロイ用）

**取得方法：**
1. [Vercel](https://vercel.com)にアクセス
2. 「Sign Up」→ GitHubアカウントでログイン（推奨）
3. 無料アカウントでOK

**費用：** 完全無料（一時的な用途なら問題なし）
